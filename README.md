# HR-Analytics-Dashboard-Power-BI

## Overview

This project presents an interactive HR Analytics Dashboard developed in Power BI to help organizations monitor workforce performance, employee retention, and workforce demographics through data-driven insights.

The dashboard consists of three main sections:

* **Overview:** Workforce snapshot and key HR metrics.
* **Retention:** Employee retention and turnover analysis.
* **Insights:** Demographic, salary, and workforce distribution analysis.
________________________________________
## Dashboard Pages

### 📊 Overview
 

Provides a high-level view of the workforce, including:

* Total Employees
* Active Employees
* Exited Employees
* Average Salary
* Workforce Distribution
* Gender & Demographic Overview

### 📈 Retention

Focuses on employee retention and turnover trends:

* Retention Rate
* Turnover Rate

### 💡 Insights

Delivers deeper workforce analysis:

* Salary Distribution
* Gender Disparity Analysis
* Ethnicity Analysis

* Job Title Analysis
